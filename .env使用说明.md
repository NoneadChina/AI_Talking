# .env文件使用说明

## 配置方法

1. **创建.env文件**：在程序可执行文件（Chat2Chat.exe）所在目录下创建一个名为`.env`的文件

2. **配置环境变量**：在.env文件中添加以下配置项（根据需要设置）：

```
# Ollama API设置
OLLAMA_BASE_URL=http://localhost:11434

# OpenAI API设置
OPENAI_API_KEY=your_openai_api_key

# DeepSeek API设置
DEEPSEEK_API_KEY=your_deepseek_api_key

# 聊天系统提示词设置
CHAT_SYSTEM_PROMPT=请使用简体中文回答问题

## 注意事项

1. **文件位置**：.env文件必须放在与Chat2Chat.exe相同的目录下

2. **变量名称**：必须使用正确的变量名称（如`OLLAMA_BASE_URL`而不是`OLLAMA_URL`）

3. **格式要求**：每个配置项使用`变量名=值`的格式，等号前后不要有空格

4. **注释**：以`#`开头的行是注释，不会被程序读取

5. **默认值**：如果某些配置项没有在.env文件中设置，程序会使用内置的默认值

## 示例.env文件

```
# Ollama API设置（必填）
OLLAMA_BASE_URL=http://localhost:11434

# OpenAI API设置（可选）
# OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# DeepSeek API设置（可选）
# DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## 生效方式

程序启动时会自动读取.env文件中的配置，并应用到程序中。如果需要修改配置，只需编辑.env文件并重新启动程序即可。
