# AI Talking 1.1.0 发布说明

发布日期: 2026-01-07

## 更新内容

### 功能改进
- 优化了聊天历史记录管理，确保聊天记录正确保存和更新
- 修复了历史功能中删除历史记录未真正保存的问题
- 改进了聊天历史导入功能，解决了气泡嵌套问题
- 优化了历史功能中导出选中功能，确保导出格式符合加载要求

### 错误修复
- 修复了Ollama API连接错误消息显示问题
- 修复了历史功能中导出全部功能的文件名格式问题
- 修复了聊天功能中加载历史按钮崩溃问题
- 修复了历史功能中清空所有按钮缺少翻译键的问题

### 性能优化
- 优化了聊天历史记录的存储和加载性能
- 改进了线程管理，减少资源占用
- 优化了UI渲染，提高响应速度

### 其他改进
- 更新了版本号到1.1.0
- 完善了文档和审计报告
- 改进了代码质量和可维护性

## 兼容性

- 与之前版本完全兼容
- 支持Windows、macOS和Linux平台
- 推荐使用Python 3.12+运行

## 安装方法

1. 从GitHub Releases下载最新安装包
2. 运行安装程序，按照提示完成安装
3. 启动AI Talking应用
