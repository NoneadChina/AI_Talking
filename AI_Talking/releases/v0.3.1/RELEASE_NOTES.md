## 安装方式

### 从安装包安装
1. 下载最新版本的安装包：AI_Talking_Setup.exe
2. 双击安装包，按照提示完成安装
3. 安装完成后，从开始菜单或桌面快捷方式启动应用

### 从源代码运行
1. 克隆仓库：`git clone http://git.corp.nonead.com/tonyke/ai_talking.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行应用：`python src/main.py`
